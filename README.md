#  Transport Operations Dashboard – Power BI Project

##  Description
A Power BI Transport Operations Dashboard analyzing fleet performance using on-time delivery %, fuel efficiency trends, maintenance cost, delivery status, and AI visuals (Q&A, Key Influencers, Decomposition Tree) to optimize routes and improve fleet usage.

---

##  Key Features

###  On-Time Delivery Performance
- Bar chart showing delivery efficiency by destination  
- Highlights delays, early deliveries, and route performance  

###  Fuel Efficiency Trend
- Line chart visualizing fuel usage over time  
- Helps identify vehicles with rising fuel consumption  

###  Cost Insights
- **Average Cost per Km** (DAX measure)  
- Pie chart for **vehicle-type maintenance cost comparison**  

###  AI Visuals
- **Q&A Visual:** Natural language query “Average cost per km by vehicle type?”  
- **Key Influencers:** Factors impacting delivery status (distance, vehicle type, driver)  
- **Decomposition Tree:** Cost per km breakdown by maintenance cost, distance, vehicle type  

---

##  Purpose
The dashboard helps transportation teams improve routing, monitor vehicle performance, reduce operational costs, and make data-driven decisions.

---

##  Tools Used
- Power BI Desktop  
- DAX Measures  
- Data Modeling (Trip Data + Vehicle Master)
